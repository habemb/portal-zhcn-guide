# portal-zhcn-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面本身不依赖任何特定域名或外部网站，主要以静态形式提供内容浏览与访问。仓库旨在整理和保留一系列相对完整的页面结构，便于日常查阅或本地使用。

## 目录结构

```
portal-zhcn-guide/
├── README.md            # 本说明文件
├── index.html           # 仓库索引页（可选）
└── pages/               # 归档页面存放目录
    └── *.html           # 各独立 HTML 页面
```

- **pages/**：存放所有归档的独立 HTML 页面。每个页面均保持完整结构，可单独打开使用。
- **index.html**（可选）：可提供一个简单的索引页，方便在 GitHub Pages 上浏览页面列表。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，包含完整的前端结构（HTML、CSS、JavaScript 等），不依赖外部域名或特定服务。
- 页面内容可能涵盖信息展示、工具界面、说明文档等，具体用途因页面而异。
- 归档的页面可能会随时间更新或补充，但原有页面一般不做修改，以保证存档的稳定性。

## 维护说明

- 本仓库由维护者不定期更新，新增或调整页面时会同步更新索引页（如有）。
- 欢迎通过 Issue 提出页面归档建议，但请注意不涉及具体域名或诱导内容。
- 不建议在本仓库中存放过于敏感或违反平台政策的页面内容。
- 如需自行部署到 GitHub Pages，可参考 GitHub 官方文档进行配置。
